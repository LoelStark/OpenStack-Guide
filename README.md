# OpenStack Study Guide (CLI + Lab Commands)

This repository contains a structured study guide for learning and revising **OpenStack** concepts and practical lab commands used in a virtualized cloud environment.

It is based on guided lab work covering setup, identity management, networking, compute operations, storage, and troubleshooting inside an OpenStack environment.

---

## 📘 Overview

This guide is designed to help students understand and practice key operations in an :contentReference[oaicite:0]{index=0} environment, including:

- Virtual machine lifecycle management
- Identity (users, projects, roles)
- Networking (subnets, routers, floating IPs)
- Compute instances
- Security groups and SSH access
- Block storage (Cinder)
- System troubleshooting
